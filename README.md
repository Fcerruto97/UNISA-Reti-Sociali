# Progetto di corso per l'esame di Reti Sociali

Implementazione in Python dell'algoritmo Targeting with Partial Incentives (TPI).

## Introduzione

Lo scopo di questo studio è quello di valutare il costo di un'operazione di viral marketing di un nuovo prodotto tra gli
studenti dell'università di Harvard. Al fine di diffondere un nuovo prodotto A in tutta la rete, si vuole individuare
l'incentivo da corrispondere ad ogni studente affinché adotti A. In particolare, il nostro studio è volto a comprendere
come variano i costi per diffondere il nuovo prodotto A al variare della threshold.
